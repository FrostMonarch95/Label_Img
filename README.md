(1)PyQt5
https://pypi.org/project/PyQt5/#files

(2)cv2
https://pypi.org/project/opencv-python/#files

(3)gdal
https://pypi.org/project/GDAL/#files

(4)scikit-image
https://pypi.org/project/scikit-image/#files

python Main.py
